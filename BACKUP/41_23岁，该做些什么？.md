<p>记得23岁的我，研究生毕业来到魔都外企开始正式工作。</p>
<p>盘点一下西班牙前锋们23岁都做了些什么。</p>
<p>23岁的劳尔，欧洲金童，金球奖排名第二。西甲欧冠金靴，拿下西甲冠军，欧冠四强。</p>
<ul>
<li>10-11 皇家马德里 西甲冠军，西班牙本土最佳球员，西甲24球金靴，欧冠四强7球金靴，西班牙国家队欧洲杯止步8强打入1球。</li>
</ul>
<p>23岁的托雷斯，2200万镑加盟利物浦，金球奖／世界足球先生都是第3位。</p>
<ul>
<li>07-08赛季 利物浦 33场24球3助攻，国家队满24岁后欧洲杯决赛破德国登顶欧罗巴。</li>
</ul>
<p>23岁的比利亚，效力西甲球会萨拉戈萨。凭借优异表现进入国家队。</p>
<ul>
<li>04-05 萨拉戈萨 西甲15球 联盟杯3球 入选<strong>国家队</strong>,首秀。</li>
</ul>
<p>23岁的索尔达多，效力保级球队赫塔费，13球带领球队保级成功。</p>
<ul>
<li>08-09 赫塔菲 西甲13球 射手榜第12位，赫塔菲17名勉强保级。</li>
</ul>
<p>23岁的内格雷多，效力阿尔梅里亚，西班牙本土银靴。</p>
<ul>
<li>08-09 阿尔梅里亚 西甲19球 射手榜第6位</li>
</ul>
<p>23岁的莫拉塔，效力意甲横行的尤文图斯，代表西班牙止步欧洲杯16强。然后回归皇家马德里。</p>
<ul>
<li>15-16 尤文 意甲7球，意杯2球，欧冠2球（欧冠表现出色）。<strong>意甲冠军。</strong>欧洲杯前友谊赛2球，<strong>欧洲杯</strong>小组赛3球，欧洲杯16强出局。（国家队近5场5球，友谊赛2球韩国，欧洲杯小组赛3球对土耳其梅开二度，对克罗地亚打入一球。）</li>
</ul>
<p>23岁的莫伦特斯应该是拿到了欧冠冠军。</p>
<p>23岁生日的帕科3000万欧元加盟巴塞罗那。在22岁的年纪刚刚错过了欧洲杯，现在23替补席上度过最美好的年华，还是抓住俱乐部与国家队替补的时间，创造属于自己的23岁。大家拭目以待吧，不想一直在替补席上嗑瓜子，就拿出点表现来吧。</p>
<p>不得不说在这个年纪去做板凳球员，是一个看上去不太好的决定，希望你能运气好点抓住也许只有2000分钟不到的上场时间。</p>
<p>前面几个赛季15球左右的表现，相对进入了瓶颈期，也许MSN能让你长点球，也许也能为你多做点球，自求多福，努力奋斗吧。</p>
